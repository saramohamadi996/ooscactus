## About This Project

This project is a Learning Management System developed with Laravel framework in modular structure. Each feature in this application developed in an isolated module, such as:

- Dashboard Module.
- Users Module.
- Courses Module.
- Lessons Modules
- Categories Module.
- Role and Permissions Module.
- Discounts Module.
- Tickets Module,
- Payments Module,
 etc...
